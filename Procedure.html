Procedure

Aim:

To create the 3D Assembly of Flange couplings

Hardware Required:

  1. CPU with Pentium IV processor.
  2. A colour monitor with highest 32-bit colour display and with screen resolution 1024 by 768 pixels.
  3. A scroll mouse.

Software Required:

  1. Windows operating system
  2. Solidworks 3-D Modelling Software

Procedures : -

  • Log on the system.
  • Open the 3d modeling cad software
  • Create the new part.
  • Generate the switches using entities and tool like circle, trim, move, etc.
  • Edit the entities if necessary.
  • Using the features like pad, fillet, mirror, revolve, etc.
  • Save the part.
  • Similarly open a new part and complete the second part and save it.
  • Similarly, all parts are created and save
  • Open the assembly part then open the all parts one by one.
  • Assemble the parts by using the tools.
  • Edit and save if necessary.
  • Exit the CAD software.
  • Log off the system.

Result:

Thus, the 3D assembly of the Flange Coupling has been created on the software Solidworks with accurate dimension and with all respects.
